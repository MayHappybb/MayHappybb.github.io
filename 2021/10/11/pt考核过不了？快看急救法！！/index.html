pt考核过不了？快看急救法！！

最近有大佬给俺发了个皇后的💊，据说考核很难过，所以想到用华为白嫖的服务器来整一整，可以实现无实体设备过考核

1.白嫖华为云服务器

注册一个华为云账号，登录后在主页点击免费试用：

以下三种各有好处，这里选择第一种，可以白嫖一个月：

点击进入后，在配置页面选择系统为Ubuntu18.04，随后设置登录密码即可完成购买。

2.安装qBittorrent

在华为云控制台找到刚刚白嫖的HECS服务器，查看其分配的公网IP地址，随后同时按win和R打开运行，输入cmd打开命令提示符，输入以下命令，注意将xxx.xxx.xxx.xxx替换为ip地址：

ssh root@xxx.xxx.xxx.xxx

随后输入预先设置的密码即可连接到服务器，接着安装qBittorrent-nox，此应用安装后运行后即可启动Web UI，便于无图形桌面的服务器使用，命令如下：

$ sudo apt-get install qbittorrent-nox

安装完成后只需输入qbittorrent-nox即可启动，但为了防止ssh窗口关闭后进程自动关闭可使用screen命令：

$ screen		//创建screen终端
$ qbittorrent-nox		//启动qBittorrent

这样即使关闭窗口也不会杀掉进程，下次若要进入此窗口，使用以下命令：

$ screen -ls		//查看已创建的screen终端

3.考核任务详解

如下图所示，考核任务一般包含五项：上传量、下载量、魔力值、分享率、做种时间，下面分别讨论如何达成。

上传量：上传量可以说是考核中最难达成的，但很多同学没发现，一般来说上传量是可以使用魔力值进行兑换的：

可以看到，只用1000魔力值就可以兑换1G的上传，因此如果很难抢到新种子刷上传的话可以将上传量考核转化为魔力值考核，如何刷到这么多魔力值，下方会介绍。

下载量：没啥好说的，下载量是最简单的考核，下载就完事了。

魔力值：如果将所有上传都使用魔力值兑换的话，一共需要4800魔力值，再加上考核要求的500魔力值，总共只需要5300点，保留一点余量，如果平均到25天达成的话，每小时只需要5400÷(25×24)=95400\div(25\times24)=95400÷(25×24)=9，也就是说每小时9点魔力值就可以达成上传和魔力值要求

分享率：即上传量：下载量，只要满足考核条件上传和下载都为5G，再加上系统初始赠送的1G上传，分享率即可≥1\geq1≥1。

做种时间：保留一点余量，如果平均到25天达成500天做种时间的话，简单计算500÷25=20500\div 25=20500÷25=20，每天要做种20天，也就是说做种20个以上即可，非常简单。

总结：多搞点魔力值就能轻松过考核！！

4.刷魔力值

留意到魔力值赚取规则：

因此想要达到每小时9点魔力值，只需要90个100M的种子做种就可以实现，或者45个300M以上的种子，这样一来做种时间也能轻松达成。同时，由于不想浪费下载量，因此全部选择Free的种子进行做种，可以使用以下方法寻找满足要求的种子：

  • 先按种子大小由小到大进行排序，点击下方标注的图标两次即可：

  • 接着向后翻两页，然后观察网址，看起来像这个样子：
https://open.cd/torrents.php?inclbookmarked=0&incldead=1&spstate=0&option-torrents=0&&sort=5&type=asc&page=2
  • 然后更改最后一个数字，这个数字表示页码，改大一点直至找到大于100M的种子，然后搜寻Free标记的种子添加到刚刚白嫖的服务器中进行下载，直到满足每小时魔力值即可。

注意

  • 记得在qBittorrent中将同时做种数量和最大活动种子数量调大一点
  • 一般Free的种子都有人下，再加上白嫖服务器具有公网ip，因此多多少少会有一点上传量
  • 由于白嫖的服务器带宽只有1Mbps，因此要注意Free的时限，将快过期的种子优先下载
