免费节点快速获取方法

免费节点快速获取方法

序言

在youtube上订阅了“破解资源君”，每天都会发布视频并在配文处贴上节点链接地址，然而每次都需要去视频下方拿链接好麻烦，就想着写个脚本爬一下，结果观察发现下载地址发现为github仓库内的文件地址，芜湖，都不用爬了，直接克隆仓库就行，巴适得很。

操作

直接克隆仓库：

$ git clone https://github.com/pojiezhiyuanjun/freev2.git
#或者
$ git clone git@github.com:pojiezhiyuanjun/freev2.git

每天只要pull以下就能同步远程仓库：

$ git pull

然后直接搜索节点文件即可，文件以日期命名，如今天是10月20日，则搜索1020.zip就可以了。
