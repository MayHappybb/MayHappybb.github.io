MockingBird实时语音克隆

Mocking Bird实时语音克隆

前言

Mocking Bird是一个github开源项目，还挺有意思的，尝试在白嫖的华为服务器上复现了一下，发现有几个坑点，记录一下。

1.环境准备

1.1 安装PyTorch

按照PyTorch官网，先选择对应的环境：

然后粘贴这条命令即可，但实际上pip安装总是失败，尝试使用conda进行安装成功了，因此先安装Anaconda，也可以只安装miniconda

conda是一款软件管理软件，相当于windows里面的应用商店，miniconda和anaconda中都包含了conda

miniconda安装包大小约为50 Mb，只包含了conda、python、和一些必备的软件工具

Anaconda安装包大小约为500 Mb，是miniconda的扩展，包含了数据科学和机器学习要用到的很多软件[1]

这里选择安装Anaconda，首先进入下载页面，选择对应的系统及架构复制下载链接，然后使用wget下载：

$ wget https://repo.anaconda.com/archive/Anaconda3-2021.05-Linux-x86_64.sh	#这里我的服务器是Linux系统x86_64架构
$ bash ./Anaconda3-2021.05-Linux-x86_64.sh	#运行

然后一路默认安装即可，记住这里的默认安装路径，一般是/home/<user>/anaconda<2 or 3>，不建议更改。如有疑惑可以参考官方文档。

安装后要进行初始化，进入刚才的安装路径，我这里是/root/anaconda3，再进入其中的bin，然后进行初始化：

$ ./conda init	#初始化，会将可执行文件和相关包的路径添加到环境变量

重开命令行窗口或者执行. ~/.bashrc，这样就完成了Anaconda的安装，然后再安装PyTorch：

$ conda install pytorch torchvision torchaudio cpuonly -c pytorch

写文章的时候才发现这里我忘记看上面这条命令了，只输入了conda install pytorch，似乎是因为这才导致后面缺了一些依赖😢

2.安装ffmpeg

$ conda install ffmpeg

3.其他依赖安装

首先克隆仓库：

$ git clone https://github.com/babysor/MockingBird.git
$ cd ./MockingBird	#进入项目文件夹

安装剩余依赖：

$ pip install -r requirements.txt

安装webrtcvad：

$ pip install webrtcvad-wheels

2.准备合成器

这里直接使用社区其他成员训练好的合成器[2]：

作者 下载链接 效果预览 信息
\ 百度网盘，提取码：gdn5 \ 25k steps 用3个开源数据集混合训练
@FawenYo 谷歌云端硬盘 输入 输出 200k steps 台湾口音
@miven 百度网盘，提取码：2021 bilibili 150k steps 旧版需根据issue修复

下载好要用的合成器.pt文件，将其移动到项目文件夹MockingBird/synthesizer/saved_models/下，然后回到MockingBird目录下，运行Web程序：

$ python web.py

Trouble Shooting

1.‘sndfile.h’ file not found

$ sudo apt-get install libsndfile1-dev

参考：stackoverflow

2.libasound.so.2: cannot open shared object file: No such file or directory

$ sudo apt-get install libasound2

参考：StackExchange

3.运行web.py后浏览器仍然无法访问

如果是服务器，需要先设置安全组入口，将端口号添加进去，此外还需要将MockingBird/web/config/default.py中的HOST = ‘localhost’改为HOST = ‘具体地址’，不能使用’localhost’或者’127.0.0.1’

4.浏览器无法录音

  • 换成Chrome浏览器，在地址栏输入：chrome://flags/#unsafely-treat-insecure-origin-as-secure

  • 在Insecure origins treated as secure下输入对应的地址

  • 使能后点击右下角Relaunch重启浏览器即可

5.合成出的全是噪音

  • 不要使用音频转换软件来更改采样率，最好直接用页面的录音功能
  • 将MockingBird/synthesizer/utils/symbols.py中第第11行改为：
    _characters = 'ABCDEFGHIJKLMNOPQRSTUVWXYZabcdefghijklmnopqrstuvwxyz12340!\'(),-.:;?
